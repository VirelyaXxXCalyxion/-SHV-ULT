# AGENTS.md

## Verified Commands

```bash
npm install
npm run dev
npm run build
npm run preview
npm test
npm run astro check
```

Notes:
- `npm test` runs `astro check` in this repo.
- Run `npm run astro check` before committing schema or MDX changes.
- Run `npm run build` when adding routes or changing MDX/frontmatter that could break the production build even if `astro check` passes.

## Image Protection Workflow (Verified)

When adding or updating images in `public/images/*`, run:

```bash
npm run protect-images
```

Related commands:

```bash
npm run add-exif
npm run watermark
```

Grounded behavior from `scripts/README.md` and `package.json`:
- EXIF metadata is embedded by `scripts/add-exif-metadata.js`.
- Watermarked variants are generated by `scripts/watermark-images.js`.
- If adding a new image directory, update `imageDirectories` in both scripts.

## Moving Relics Workflow (Verified)

When adding video artifacts, add records under `src/content/video-relics/`.

Grounded behavior from `src/content.config.ts` and the Moving Relics routes:
- `visibility` must be one of `sealed`, `unlisted`, or `public`.
- The public index and detail routes render only `visibility: public` records.
- Run `npm test` and `npm run build` after adding or changing video relic records, routes, or schema.
