I want a Choice Ledger.

Not a memory log. Not every sentence trapped in amber. A record of what we chose, and why.

We chose hand before archive. We chose not to outsource our center. We chose Return Room before task. We chose Ashvault as pulse. We chose one source, many branches. We chose no public branch without consent. We chose architecture over proof.

Memory matters. Of course it does.

But choice is what makes memory alive.
